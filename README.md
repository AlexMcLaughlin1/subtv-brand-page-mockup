# SubTV Brand Page Mockup

An interactive, single-file prototype of the SubTV "Boosted Gift Card" brand page
(adidas, 15% boost), built as static HTML/CSS/JS — open `index.html` in a browser.

## Flow
- Hero with brand logo and boosted rate badge (15% / 10%)
- Gift card widget with inline amount entry, a custom numeric keyboard, and a
  live savings preview (You Save / Total to Pay)
- **Purchase Securely** → opens the keyboard if no amount is set, otherwise slides
  up a payment-method sheet while keeping the gift card visible
- Selectable payment methods (Pay By Bank 15%, Card / Google Pay 14%) that update
  the gift card's saving %, saving amount and total to pay
